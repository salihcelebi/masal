## Üst Düzey Dosyalar

Üst düzey dosyalar uygulamayı yapılandırmak, bağımlılıkları yönetmek, middleware çalıştırmak ve ortam değişkenlerini tanımlamak için kullanılır.

|                                                                                             |                                     |
| ------------------------------------------------------------------------------------------- | ----------------------------------- |
| **Next.js**                                                                                 |                                     |
| [`next.config.js`](/docs/app/api-reference/next-config-js)                                  | Next.js yapılandırma dosyası        |
| [`package.json`](/docs/getting-started/installation#manual-installation)                    | Proje bağımlılıkları ve scriptler   |
| [`instrumentation.ts`](/docs/app/building-your-application/optimizing/instrumentation)      | OpenTelemetry ve instrumentation     |
| [`middleware.ts`](/docs/app/building-your-application/routing/middleware)                   | Next.js istek middleware'i          |
| [`.env`](/docs/app/building-your-application/configuring/environment-variables)             | Ortam değişkenleri                  |
| [`.env.local`](/docs/app/building-your-application/configuring/environment-variables)       | Yerel ortam değişkenleri            |
| [`.env.production`](/docs/app/building-your-application/configuring/environment-variables)  | Üretim ortamı değişkenleri          |
| [`.env.development`](/docs/app/building-your-application/configuring/environment-variables) | Geliştirme ortamı değişkenleri      |
| [`.eslintrc.json`](/docs/app/building-your-application/configuring/eslint)                  | ESLint yapılandırma dosyası         |
| `.gitignore`                                                                                | Yoksayılacak Git dosyaları          |
| `next-env.d.ts`                                                                             | Next.js TypeScript bildirim dosyası |
| `tsconfig.json`                                                                             | TypeScript yapılandırma dosyası     |
| `jsconfig.json`                                                                             | JavaScript yapılandırma dosyası     |
